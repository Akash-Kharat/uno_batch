Important Notes
###############

- What's the use of below CSS?

    * {
        box-sizing: border-box;
    }

    Ans: This ensures that all the elements do not exceed the width and height limitations due to padding or border.

- how to make a div fixed onto the screen?
    1. By using  position: fixed
    2. Then position it using top, left, bottom or right properties.
    3. Control the z-axis behaviour using z-index CSS property.

- How to overlay an HTML Element over another HTML element?

    Ans: 
    1. By using position: relative for parent container and position: absolute for the HTML element that is to be overlayed.
    2. Then position it using top, left, bottom or right properties.
    3. Control the z-axis behaviour using z-index CSS property.
