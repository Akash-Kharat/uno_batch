Important Notes
################

* All the input fields must have type, id and name attributes
* To connect a label with an input field we use "for" in label with "id" in input field
* Diffrence between "required" and "disabled" attribute.
* if novalidate attribute is present in the form then all the input fields will not validated before submit.
